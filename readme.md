## Getting Started

### Prerequisites

-   SpringFramework 5.3.20
-   Maven

### Installation

1. Clone the repository

    ```bash
    git clone https://github.com/RosieOh/TeamLimked_Company_page.git
    ```

2. Install Maven Dependencies



3. Run the development server
